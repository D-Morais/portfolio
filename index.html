Página de Resturante

Controle de Finanças

O projeto foi elaborado empregando a linguagem de programação Python, com a inclusão das bibliotecas Dash e Pandas. Ele incorpora a integração de planilhas do Excel e se destina a oferecer um Dashboard de Controle Financeiro. Este dashboard interativo é projetado para apresentar de maneira dinâmica o saldo financeiro, fundamentado nas transações de entrada e saída.
